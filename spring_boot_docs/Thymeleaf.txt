Thymeleaf template engine
Thymeleaf is a mordern server-side java template engine for both web and 
standalone enviroment, capable of processing HTML, XML, javaScript, CSS and even plain text.

The main goal of Thymeleaf is to provide an elegan and highly maintainable way of creating templates

Mostly used to generate html views for web applications
